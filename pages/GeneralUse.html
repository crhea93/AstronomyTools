General Tools

Combine Regions

This file has the simple task of combining several ds9 regions into a single file.
It currently supports both the addition and subtraction of regions.

Merge

The script combines several Chandra OBSIDs after they have been reprocessed

MonoEnergy

Script with calculates the monochromatic energy for a given energy range and region using weighted arf files

Precursor

In my opinion, this is one of the more useful tools. It simply creates fits files from regions of interest and subsequently runs specextract on it.
